Monitoramento de Energia

Sobre o Projeto

Este projeto demonstra como o Arduino pode ser usado para monitorar e gerenciar o consumo de energia. O sistema coleta dados de tensão, corrente e potência, enviando-os para um servidor web em intervalos regulares. O objetivo é facilitar o acompanhamento, controle e auditoria do consumo energético, tornando o uso mais eficiente e seguro.

  • Os dados enviados incluem código do equipamento, data, hora, tensão, corrente, potência e status.
  • O retorno da API informa os dias e horários permitidos de uso, potência máxima fora do horário e intervalo de atualização.
  • Os gráficos exibem a variação dos principais parâmetros elétricos ao longo do tempo.

Ideal para projetos de automação residencial, laboratórios, indústrias e qualquer ambiente que exija controle inteligente do consumo de energia.
